An Open-Source DeepSeek API Balance Query Application

In plain English: I created an open-source application for querying DeepSeek API key account balances. It was primarily written using DeepSeek v4 Flash, and I used a rather aggressive approach (i.e., if the AI suddenly goes off the rails, it could delete all repository files at any time). The same content will be published later on my personal blog.

Main Content:

Developed with DeepSeek v4 Flash — an app that allows users to directly query and batch query account balances by filling in DeepSeek API keys, leveraging Android WebView to run the HTML files embedded in the APK. The HTML files inside the app can be extracted and used in browsers such as Via, Chrome, Firefox, or the system's built-in browser. Consideration will be given to posting them on my personal blog in the future.

The project is primarily HTML-based, with Java as a supporting language. However, I'm not sure why DeepSeek v4 Flash and DeepSeek v4 Pro insist on using Java instead of what I personally consider to be the better choice — Kotlin.

During development, I adopted an aggressive approach: giving DeepSeek v4 Flash SSH and GitHub keys, allowing the LLM full control over my personal GitHub repositories. Yet DeepSeek v4 Flash successfully completed the task and uploaded everything to GitHub as open source.

Currently, due to doubts raised by group members, it has been open-sourced on GitHub (though I personally don't see the necessity of open-sourcing it).
